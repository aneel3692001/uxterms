# Version 1.0.0

**Release Date**: 2025-11-27

## Summary

This is the initial MVP release of the UX Terms platform. It includes all core learning features and the gamification engine.

## Key Features

1. **Gamification Engine**: Tracks user progress locally without a backend.
2. **Design System**: Implements a premium Glassmorphism aesthetic with "Cult UI" vibes.
3. **Content**: Pre-seeded with 10+ glossary terms, 2 quiz categories, and 4 design challenges.

## Known Issues

- Anti-cheat is client-side only and can be bypassed.
- Progress is lost if browser cache is cleared.
